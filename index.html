Quadlek: a customizable slack bot framework

Getting Started

Quadlek is a framework written in Go for creating a Slack bot that can do your bidding. It is stable, highly concurrent, and customizable via an easy to use plugin interface. Using quadlek, you can easily enable your users to increase(or decrease) productivity with chatops and other fun.

Quadlek ships with a number of plugins including:

  • karma - Let users give or take points to other users
  • Github - Create, view, and comment on Github issues
  • random - Roll dice, choose from a list of options
  • Spotify - Lets users share what they are listening to and can store any songs linked to into a public shared playlist
  • Twitter - Automatically publish tweets for tracking news, service status updates, etc

Some other plugins that have been developed for Quadlek:

  • comics - Generate a comic based on recent chat and upload it to Imgur
  • cointip - Tip users with Bitcoin using the Coinbase API
  • archive - Archive a URL to IPFS for historical viewing

Installation

To begin using Quadlek to build your new Slack bot, you’ll need to install the library to your Go path:

$ go get -u github.com/jirwin/quadlek/quadlek
